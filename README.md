🎨 CSS Gerador com IA

Um gerador de código HTML + CSS utilizando IA.  
Você descreve o que quer e a aplicação cria o código automaticamente com preview em tempo real ⚡

🚀 Funcionalidades  
  
✍️ Entrada de texto para descrever o que deseja  
🤖 Integração com IA (Groq API)  
💻 Geração automática de HTML + CSS  
👀 Preview em tempo real do resultado  
🎨 Interface simples e moderna  

🛠️ Tecnologias utilizadas  
HTML5  
CSS3  
JavaScript   
API de IA (Groq)  

- Para configurar sua API KEY 

No arquivo scripts.js, substitua:  
let chave = "Coloque_sua_chave_aqui"  
E também:  
"Authorization": "Bearer Coloque_sua_chave_aqui"  
Pela sua chave da API da Groq.
